# Cobblemoon — Modpack (packwiz)

Auto-updating modpack for the **Cobblemoon** server (Cobblemon & Ad Astra, Fabric 1.21.1).
Mod/resourcepack jars are hosted externally; this repo holds the packwiz manifest + configs.

## Install (Prism Launcher — auto-updates on launch)
1. Install **Prism Launcher** (free) and add your Minecraft account.
2. **Add Instance → Import → from packwiz** (or use the packwiz bootstrap pre-launch command) with this URL:
   `https://raw.githubusercontent.com/Anthonyzchen/cobblemoon-pack/main/pack.toml`
3. Launch. The pack installs Fabric 1.21.1 + all mods + configs, and pulls only changed files on every future launch.
4. Multiplayer → `azc-cobblemon.duckdns.org`. ~6–8 GB RAM. Shaders (Complementary) come pre-enabled via Iris.

Requires a legitimate Minecraft account.
